# Create Architecture Workflow

Create comprehensive architecture documentation:

1. Read `_bmad/bmm/workflows/3-solutioning/create-architecture/workflow.md`
2. Analyze PRD requirements for technical decisions
3. Follow architecture design methodology
4. Document system components, patterns, and decisions
5. Save architecture document to project documentation
