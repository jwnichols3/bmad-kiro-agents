# Create PRD Workflow

Create a comprehensive Product Requirements Document:

1. Read `_bmad/bmm/workflows/2-plan-workflows/create-prd/workflow-create-prd.md`
2. Follow PRD creation methodology
3. Transform product brief into detailed requirements
4. Structure PRD according to workflow template
5. Save completed PRD to project documentation
