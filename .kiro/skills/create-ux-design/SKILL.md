# Create UX Design Workflow

Create comprehensive UX design documentation:

1. Read `_bmad/bmm/workflows/2-plan-workflows/create-ux-design/workflow.md`
2. Analyze PRD for user experience requirements
3. Follow UX design methodology and patterns
4. Document user flows, wireframes, and design decisions
5. Save UX design document to project documentation
